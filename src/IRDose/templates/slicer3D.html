How to create source and target organs with Slicer3D
Courtesy of Jonathan Tranel

  1. On up-left corner click on "DCM" > "Import" > "load".
  2. In "Modules" select "Segment Editor" -> "Add segment".
  3. On "Effects", select "Draw".
  4. Draw the desired region and press Enter to add. It is also possible to erase mistakes by selecting the "Erase" brush.
  5. Repeat the previous action over all slices that contains the desired volume/organ.
  6. Once the whole organ is covered, click on "Create surface".
  7. On "Modules" select "Segmentations" -> "add label map" -> "Copy/move segment" --> " > "
  8. On "Modules" select "Mask Scalar Volume"

    1. Input Volume: the DICOM file
    2. Mask volume: the created volume
    3. Masked volume: "Create new as" -> Name your volume.
    4. Label value: 1
    5. Replace value : 0
    6. Click on "Apply"

  9. On "Modules" select "Subject hierarchy"

    1. Right click on "Scene" -> "Create new subject"
    2. Right click on "New subject" -> "Create new child study"
    3. Glide the volume created at 8.iii to "New Study"
    4. Right click on "New subject" -> "Export to DICOM..." -> Choose the destination folder
